# 小红书 (Little Red Book)

模仿"小红书"风格的原生纯前端项目，技术栈：原生 HTML、JS、CSS，后端数据采用 localStorage。

## 项目预览

访问 http://localhost:8080 查看项目

## 功能特性

### UI 风格
- 极简主义设计
- 主色调：小红书红 (#ff2442)
- 适配移动端响应式布局

### 页面结构

#### 1. 首页 (Feed)
- 双列瀑布流布局
- 每个卡片包含：封面图、标题、作者头像、昵称、点赞数
- 支持点赞、收藏、评论、分享操作

#### 2. 发布页面
- 图片上传预览
- 标题和内容输入
- 发布新笔记

#### 3. 我的页面 (Profile)
- 背景图 + 头像 + 基本信息
- 关注/粉丝/获赞统计
- 三个标签页：
  - 作品（九宫格展示）
  - 已点赞
  - 收藏

#### 4. 关注/粉丝列表
- 点击关注数或粉丝数弹出用户列表
- 支持关注/取消关注操作

### 核心功能

#### 登录系统
- 弹窗式登录界面
- 默认账号：admin / 123456
- 用户信息存储在 localStorage

#### 点赞功能
- 点击爱心图标变红
- 点赞数实时增加
- 数据持久化存储

#### 收藏功能
- 五角星收藏按钮
- 收藏内容在"我的-收藏"页签展示

#### 评论系统
- 每条笔记支持评论
- 发表评论
- 查看评论列表
- 点赞评论

#### 分享功能
- 模拟分享按钮
- 底部操作栏（分享给好友、复制链接、微信、微博）

## 技术栈

- **HTML5**: 语义化标签，移动端适配
- **CSS3**: Flexbox/Grid 布局，动画效果，响应式设计
- **JavaScript (ES6+)**: 模块化开发，localStorage 数据持久化
- **数据存储**: localStorage

## 快速开始

### 1. 启动本地服务器

```bash
# 使用 Python
python -m http.server 8080

# 或使用 Node.js
npx serve .

# 或使用 VS Code Live Server 插件
```

### 2. 访问项目

打开浏览器访问 http://localhost:8080

## 使用说明

### 登录
1. 点击"我的"页面
2. 点击头像或"点击登录"文字
3. 输入账号：admin，密码：123456
4. 点击登录按钮

### 浏览内容
- 首页自动加载瀑布流内容
- 上下滑动浏览更多

### 互动操作
- **点赞**: 点击卡片上的爱心图标
- **收藏**: 点击卡片上的星星图标
- **评论**: 点击评论图标打开评论弹窗
- **分享**: 点击分享图标打开分享选项

### 发布笔记
1. 点击底部导航栏的"+"按钮
2. 上传图片（可选）
3. 填写标题和内容
4. 点击发布

### 查看关注/粉丝
1. 进入"我的"页面
2. 点击关注数或粉丝数
3. 在弹窗中查看列表
4. 点击关注/取消关注按钮

## 数据存储

所有数据存储在浏览器 localStorage 中，包括：
- `currentUser`: 当前登录用户信息
- `posts`: 帖子数据
- `likedPosts`: 已点赞的帖子ID列表
- `collectedPosts`: 已收藏的帖子ID列表
- `following`: 关注列表
- `followers`: 粉丝列表

清除浏览器数据将会重置所有内容。

## 开发说明

### 代码结构

```javascript
// 主要模块
Storage     // localStorage 操作封装
MockData    // 模拟数据生成
AppState    // 应用状态管理
Utils       // 工具函数
Render      // 渲染函数
Actions     // 操作函数
```

### 添加新功能

1. 在 `Actions` 对象中添加操作函数
2. 在 `Render` 对象中添加渲染函数
3. 在 `bindEvents` 中绑定事件
